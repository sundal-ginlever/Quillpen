# 📝 Quillpen (inkcanvas) 사용자 가이드

Quillpen은 무한한 캔버스 위에 메모, 스케치, 스프레드시트, 이미지를 자유롭게 배치하고 연결할 수 있는 단일 페이지 웹 애플리케이션입니다.

## 🚀 시작하기

1. **로컬 모드**: `index.html`을 브라우저에서 열면 즉시 오프라인 로컬 모드로 동작합니다. 데이터는 브라우저의 `localStorage`에 자동 저장됩니다.
2. **클라우드 모드**: `supabase_setup.sql`을 Supabase에 적용하고 `index.html` 상단의 URL/Key를 변경하면 실시간 동기화와 클라우드 저장이 활성화됩니다.

---

## 🛠️ 제공되는 위젯 종류

1. **📝 메모 위젯 (M)**: 텍스트 입력, 6가지 배경색 변경, 자유로운 리사이즈.
2. **🎨 스케치 위젯 (S)**: 자유 드로잉, 8색상/4굵기, 지우개 지원. (모바일 터치 완벽 지원)
3. **📊 스프레드시트 위젯 (T)**: 엑셀과 유사한 표 계산기. 수식 지원, 행/열 크기 조절, 단축키 지원.
4. **🖼️ 이미지 위젯 (I)**: 드래그 앤 드롭 업로드 지원.

## 🧮 스프레드시트 수식 목록

수식바에 `=`을 입력하여 사용합니다.

| 수식 | 설명 | 예시 |
|------|------|------|
| `SUM(범위)` | 합계 | `=SUM(A1:C5)` |
| `AVG(범위)` | 평균 | `=AVG(A1:A10)` |
| `MIN(범위)`, `MAX(범위)` | 최솟값, 최댓값 | `=MAX(B1:B10)` |
| `COUNT(범위)` | 숫자가 있는 셀 개수 | `=COUNT(A1:C5)` |
| `COUNTA(범위)` | 비어있지 않은 셀 개수 | `=COUNTA(A1:A10)` |
| `PRODUCT(범위)`| 범위의 곱 | `=PRODUCT(A1:A5)` |
| `IF(조건, 참, 거짓)`| 조건문 | `=IF(A1>10, "크다", "작다")` |
| `CONCAT(값1, 값2)`| 텍스트 결합 | `=CONCAT(A1, "원")` |
| 기타 수학 | `ABS`, `SQRT`, `ROUND`, `INT` | `=ROUND(A1)` |
| 기타 텍스트 | `LEN` (문자열 길이) | `=LEN(B1)` |
| 사칙연산 | 기본 사칙연산 연계 | `=A1+B2*3` |

## ☁️ 내보내기 및 데이터 백업

단축키 `Ctrl+Shift+E`를 통해 다음 형식으로 데이터를 내보낼 수 있습니다.
- **PNG**: 스케치 선을 포함한 전체 캔버스의 고화질 스크린샷
- **JSON**: 모든 위젯과 설정이 포함된 완벽한 백업 파일 (가져오기 복원 가능)
- **CSV**: 스프레드시트의 내용을 엑셀에서 바로 열 수 있는 형식으로 내보내기
