* phase 1 스프레드시트 보완으로 phase 2 Code 재점검을 통한 문제점

  1. 스프레드시트 렌더러 — Phase 2에는 구버전(더블클릭 편집, +행/+열만 있는 단순 버전)이 그대로 들어있음. 완성된 Phase 1 버전(수식바, 범위선택, 서식, 키보드 탐색, 컨텍스트 메뉴 등)으로 교체 필요
  2. widgetData() 직렬화 누락 — 스프레드시트의 colWidths, rowHeights, cellFmt, boldCells, italicCells 가 클라우드 저장 시 제외되어 있어 다른 기기에서 열면 서식이 사라짐
  3. rowToWidget() 역직렬화 — 위 누락된 메타데이터를 복원하는 로직도 없음
  4. pan 이중 실행 버그 — pointermove 리스너가 두 개 등록되어 pan 속도가 2배로 튐
* 재점검에 따른 phase 2 보완

  FIX 1 — widgetData() 직렬화 보완
스프레드시트를 클라우드에 저장할 때 colWidths, rowHeights, cellFmt, boldCells, italicCells 가 누락되던 문제 수정. 이제 열/행 크기 조절과 서식이 다른 기기에서도 그대로 복원됩니다.

  FIX 2 — rowToWidget() 역직렬화 보완
클라우드에서 불러올 때 배열로 저장된 boldCells, italicCells를 Set으로 복원하고, 나머지 메타데이터도 기본값으로 초기화.

  FIX 3 — handleRealtimeChange() 스프레드시트 분기
다른 기기에서 스프레드시트 셀 내용이 바뀌면 단순 위치/크기 업데이트 대신 전체 리빌드 실행. 메모는 포커스 없을 때만 텍스트 업데이트.

  FIX 4 — pan 이중 실행 버그
pointermove 리스너가 두 개 등록되어 이동 속도가 2배로 튀던 문제 수정. 단일 리스너로 통합하고 lastPanPt 기반으로 정확한 델타 계산.

  FIX 5 — saveLocal() Set 직렬화
JSON.stringify는 Set을 {}로 변환하므로, localStorage 저장 전 \[...Set] 배열로 변환.

  FIX 6 — loadLocal() Set 복원
localStorage에서 불러올 때 배열을 다시 Set으로 복원하고 나머지 스프레드시트 메타데이터도 기본값 설정.

  내보내기 — PNG(캔버스 전체 스크린샷), CSV(스프레드시트), JSON(전체 백업)

  PWA — 홈 화면 설치, 오프라인 동작, 앱 아이콘

  키보드 단축키 도움말 — ? 키로 단축키 팝업

  캔버스 이름 편집 — 툴바에서 인라인 편집



  공유 링크 (Ctrl+Shift+S)

  툴바 🔗 공유 버튼 클릭 → 공유 ON/OFF 토글 → 고유 토큰이 포함된 URL 생성. ?share=토큰 으로 접근하면 로그인 없이 읽기 전용으로 캔버스를 볼 수 있습니다. 편집 UI는 자동으로 비활성화됩니다.

  내보내기 (Ctrl+Shift+E)



  PNG — 전체 캔버스를 스크린샷으로 저장. 스케치 스트로크까지 재렌더링

  JSON — 모든 위젯 데이터, 카메라, 설정을 백업 파일로 저장

  CSV — 스프레드시트 블록을 엑셀 호환 CSV로 저장 (수식은 계산값으로 출력)

  JSON 가져오기 — 백업 파일로 캔버스 복원



  PWA 앱 설치

  브라우저가 설치를 지원하면 툴바에 📲 앱 설치 버튼이 자동으로 나타납니다. 홈 화면에 추가하면 네이티브 앱처럼 전체화면으로 실행됩니다.

  캔버스 이름 편집

  툴바 중앙의 캔버스 이름을 클릭하면 인라인 편집됩니다. Enter로 확정, Esc로 취소. Supabase에 자동 저장됩니다.

  단축키 도움말 (?)

  모든 단축키를 한눈에 볼 수 있는 팝업입니다.



  Supabase 적용 순서 (3단계):



  1/ supabase\_setup.sql 을 Supabase SQL Editor에서 실행

  2/ Dashboard → Database → Replication에서 widgets 테이블 Realtime 활성화

  3/ inkcanvas-v3.html 상단 SUPABASE\_URL / SUPABASE\_ANON 값 교체



  1\. 공유 링크

  캔버스를 URL 하나로 다른 사람에게 공유하는 기능입니다. 작동 방식은 이렇습니다. 공유를 켜면 Supabase의 canvases 테이블에 랜덤 토큰(share\_token)이 생성되고, https://...?share=토큰 형태의 URL이 만들어집니다. 이 URL로 접근하면 로그인 없이도 캔버스를 볼 수 있습니다. 단, 읽기 전용이라 편집은 불가합니다. 상단에 "읽기 전용 공유 캔버스" 배너가 표시되고, 편집 UI 버튼들은 자동으로 비활성화됩니다. 이걸 가능하게 하는 건 Supabase RLS(Row Level Security) 정책으로, share\_enabled = true인 캔버스는 비로그인 사용자도 SELECT할 수 있도록 설정했습니다.

  2\. 내보내기 / 가져오기

  세 가지 형식을 지원합니다. PNG는 캔버스 전체를 이미지로 저장하는데, 단순 스크린샷이 아니라 모든 위젯의 월드 좌표를 계산해서 <canvas> 위에 직접 재렌더링합니다. 스케치 블록의 필기 획까지 그대로 재현됩니다. JSON은 위젯 데이터, 카메라 위치, 설정을 전부 담은 백업 파일로, 나중에 다시 가져오기(importJSON)로 완전 복원이 됩니다. CSV는 스프레드시트 블록을 엑셀 호환 형식으로 저장하는데, 수식이 있으면 계산된 결과값으로 출력합니다.

  3\. PWA (Progressive Web App) 설치

  브라우저가 앱 설치를 지원하는 환경(Chrome, Edge, 모바일 Safari 등)에서 접속하면 툴바에 📲 앱 설치 버튼이 자동으로 나타납니다. beforeinstallprompt 이벤트를 잡아서 버튼이 눌릴 때 브라우저 기본 설치 프롬프트를 띄우는 방식입니다. 설치하면 홈 화면에 아이콘이 생기고, 실행 시 주소창 없이 전체화면으로 동작합니다. HTML 파일 안에 data: URL 형태로 manifest를 인라인으로 넣었기 때문에 별도 서버 설정 없이도 동작합니다.

  4\. 캔버스 이름 편집

  툴바 중앙에 캔버스 이름이 표시되고, 클릭하면 바로 그 자리에서 텍스트 편집이 됩니다. Enter로 확정하면 Supabase canvases 테이블에 자동 저장되고 브라우저 탭 제목도 함께 바뀝니다.

  5\. 단축키 도움말

  ? 키를 누르면 모든 단축키를 모아서 보여주는 팝업이 뜹니다. 새로 추가된 Ctrl+Shift+S(공유), Ctrl+Shift+E(내보내기)도 포함됩니다.



  ## 4\. phase 4 개발

  Phase 4 구현 항목 — 실사용 완성도를 높이는 것들:



  Undo / Redo (Ctrl+Z / Ctrl+Y) — 모든 작업 되돌리기

  위젯 복제 (Ctrl+D) — 선택한 위젯 그대로 복사

  이미지 위젯 — 드래그 앤 드롭 또는 클릭으로 이미지 추가

  전체 맞춤 줌 (Ctrl+Shift+F) — 모든 위젯이 화면에 들어오는 줌 레벨 자동 계산

  위젯 잠금 — 선택 후 잠금 시 이동/삭제 방지

  캔버스 전체 검색 (Ctrl+F) — 메모/스프레드시트 텍스트 검색



  Phase 4 완성입니다. 구현된 6가지 기능을 정리합니다.

  Undo / Redo (Ctrl+Z / Ctrl+Y)

  모든 위젯 조작(생성, 이동, 삭제, 내용 변경)이 최대 60단계까지 스냅샷으로 저장됩니다. save()가 호출될 때마다 자동으로 스냅샷을 찍고, undo 시 이전 상태 전체를 복원합니다. 작업 결과는 하단 토스트 메시지로 표시됩니다.

  위젯 복제 (Ctrl+D)

  선택한 위젯을 오른쪽 아래 24px 오프셋으로 즉시 복제합니다. 다중 선택 상태에서도 동작하고, 복제된 위젯들이 자동으로 선택 상태가 됩니다.

  이미지 위젯 (I 키 또는 드래그로 생성)

  이미지 블록을 생성하면 드롭 존이 나타나고, 클릭해서 파일 선택하거나 이미지 파일을 직접 드래그 앤 드롭할 수 있습니다. contain / cover / fill 세 가지 표시 방식 전환이 가능합니다. 캔버스 빈 공간에 이미지 파일을 그냥 드래그해서 올려도 자동으로 이미지 위젯이 생성됩니다.

  화면에 맞춤 (Ctrl+Shift+F)

  모든 위젯의 경계 박스를 계산해서 전체가 화면에 들어오는 줌 레벨과 카메라 위치를 자동으로 설정합니다. 실행 시 화면에 미세한 플래시 효과가 나타납니다.

  위젯 잠금 (Ctrl+L)

  잠긴 위젯은 주황색 점선 테두리로 표시되고, 이동과 삭제가 차단됩니다. 다시 Ctrl+L을 누르면 해제됩니다.

  전체 검색 (Ctrl+F)

  툴바 아래에 검색 바가 열리고, 입력하는 즉시 메모 내용과 스프레드시트 셀 값을 실시간으로 검색합니다. 결과 개수가 표시되고 ↑↓ 버튼이나 Enter로 결과 간 이동하면 해당 위젯으로 카메라가 자동 이동되고 노란색 하이라이트가 표시됩니다.
